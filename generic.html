Kiber Təhlükəsizlik Kursları

Emin Savaylov tərəfindən keçirilən kiber təhlükəsizlik dərsləri həm Red Team (hücum), həm də Blue Team (müdafiə) yanaşmalarını əhatə edir. Siz yalnız nəzəri biliklərlə kifayətlənməyəcək, real laboratoriya mühitində şəbəkə hücumları, sistem zəiflikləri və mobil təhlükəsizlik ssenariləri ilə işləyəcəksiniz.

Red Team mövzularında siz şəbəkə hücumları (DoS, DDoS, MITM, ARP spoofing), sistem zəifliklərinin istismarı (buffer overflow, ROP chain, privilege escalation), mobil hücumlar (Android APK dekompilyasiyası, iOS zəiflikləri, zərərli proqram yazılması), kriptografiya hücumları (hash cracking, bruteforce, rainbow tables) və web tətbiqləri zəiflikləri (XSS, SQL Injection, CSRF, file upload bypass) üzrə biliklər qazanacaqsınız.

Hücum texnikalarında Python, JavaScript, PHP və C dili üzərində exploit yazılması, SQL sorğularının manipulyasiyası, JavaScript ilə XSS payloadlarının yaradılması, Python ilə şəbəkə skaneri və exploit skriptləri, C dili ilə low-level zəifliklərin (buffer overflow) təhlili praktiki şəkildə öyrədiləcək. Bu yanaşma sizə həm proqramlaşdırma biliklərinizi artırmağa, həm də zəifliklərin necə işlədiyini başa düşməyə imkan verəcək.

Blue Team dərslərində isə şəbəkə müdafiəsi (firewall, IDS/IPS, SIEM analizi), sistemlərin qorunması (patch management, endpoint security, antivirus və EDR alətləri), mobil təhlükəsizlik (secure coding for Android/iOS, zərərli proqramların aşkarlanması), web müdafiə üsulları (WAF, input validation, secure coding practices) və log analizi ilə hücumların aşkarlanması öyrədilir.

Bu kurs həm başlanğıc, həm də inkişaf etmiş istifadəçilər üçün nəzərdə tutulub. Sonda siz özünüzü həm ethical hacker kimi hücum texnikalarında, həm də cyber defender kimi müdafiə yanaşmalarında inkişaf etdirərək güclü bir kiber təhlükəsizlik mütəxəssisi ola biləcəksiniz.
